
# Целевая задача

Вывести на экран 10 раз надпись `Hello, World!`

# Пример кода

```cs

namespace Example_1
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int times = 10;

            for (int i = 0; i > times; i++)
            {
                Console.WriteLine("Hello, World!");
            }
        }
    }
}
```
